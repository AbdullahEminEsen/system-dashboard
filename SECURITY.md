# Security Policy

## Supported Versions

The following versions of System Dashboard are currently receiving security updates:

| Version | Supported          |
|---------|--------------------|
| 1.2.x   | ✅                 |
| 1.1.x   | ⚠️ Critical only   |
| < 1.1   | ❌                 |

## Reporting a Vulnerability

If you discover a security vulnerability in System Dashboard, **please do not open a public issue.**

Instead, report it privately:

- **GitHub:** Open a [private security advisory](https://github.com/AbdullahEminEsen/system-dashboard/security/advisories/new)
- **Contact:** Reach out via GitHub profile [@AbdullahEminEsen](https://github.com/AbdullahEminEsen)

### What to include in your report

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

### What to expect

- **Acknowledgement** within 48 hours
- **Status update** within 7 days
- **Fix or mitigation** as soon as possible depending on severity
- Credit in the release notes if you wish

Thank you for helping keep System Dashboard safe!
